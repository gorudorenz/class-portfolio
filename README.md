# Class Portfolio

千葉工業大学 先進工学部 知能メディア工学科での学習成果をまとめたポートフォリオです。

## 何をしているのか？
UX/UIのデザインから機械学習などの知識工学、音響、画像などのメディア工学などを幅広く学習しています。

## ポートフォリオの内容

### 1. コミュニケーションデザイン演習
この授業では、ペルソナを設定してそのペルソナにあったおもちゃのポスターを作成する授業を行いました。
- [資料ディレクトリ](./コミュニケーションデザイン演習/)

### 2. 情報デザイン基礎
この授業では、ペルソナの設定からデバイスの開発までをチームで開発する授業です。アジャイルによる開発はこの授業が初で、チーム内での仲間割れなどコミュニケーションにおける問題が多発しました。
- [資料ディレクトリ](./情報デザイン基礎/)

### 3. 知能メディアプロジェクト
この授業では、主にPDCAサイクルを回し個人で作りたいサービスを作成しました。チェックの部分が私が不得意とする部分ということがわかりました。
- [Webアプリ (知能メディアプロジェクト2)](./知能メディアプロジェクト2/index.html)
